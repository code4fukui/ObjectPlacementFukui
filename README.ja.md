# 空間オブジェクト

AR空間に3Dオブジェクトを配置できるサンプルアプリです。Apple Vision Proで体験できます。

## 機能
- 平面の検出と物体の配置
- 配置済みの物体の移動・削除
- 複数の3Dモデルから選択可能

## 必要環境
- Apple Vision Pro
- iOS 16.4以降

## 使い方
1. アプリを開く
2. 平面を検出する  
3. 配置したいオブジェクトを選択する
4. タップして平面上に配置する
5. 配置したオブジェクトはドラッグして移動したり、ゴミ箱アイコンをタップして削除できる
